Téléchargez la version gratuite














Synthèse des éléments techniques

La version gratuite inclue toutes les fonctionnalités et permet d’implémenter 1 version d’un processus pour un nombre illimité d’utilisateurs sur une instance. Il n’y a pas de limites de temps et elle est gratuite pour toujours.


  • Application Web .NET exécutée sur Windows Server (2008, 2012, 2016) ou Windows 10 (environnements de développement).

  • Bases de données supportées : MS SQL (2008, 2012, 2014, 2016, Express Edition) et Oracle (10g, 11g et R2, 12c).

  • Entièrement intégrée à IIS, avec support de l'authentification Windows transparente (Active Directory) intégrée (d'autres méthodes d'authentification sont disponibles).
  • Utilise la passerelle IIS SMTP ou un serveur SMTP existant pour envoyer les notifications par email.

  • La configuration comprend l'installation d'une application Web .NET et l'installation d'un service Windows.

  • Les fichiers de données d'application sont stockés par défaut sur le serveur Web.
