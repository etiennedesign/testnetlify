Poco código, infraestrctura a prueba de todo, alto rendimiento

Entregue soluciones avanzadas y basadas en procesos
con un alto nivel de configurabilidad

WorkflowGen integra un potente motor de flujo de trabajo .NET optimizado para ejecutar procesos complejos manejando un gran número de transacciones. Su módulo de administración web altamente funcional ofrece una experiencia de usuario fluida para manejar sus procesos. Esto le permitirá concentrarse en la configuración de los procesos y los puntos de integración gracias a las funcionalidades avanzadas de flujo de trabajo disponibles, como por ejemplo la gestión de delegaciones, las notificaciones personalizadas, los sub-procesos, la paralelización de actividades a través de sincronizaciones, la gestión de retrasos, las acciones del sistema (webhooks, servicios web, ensamblajes (assbemblies) .NET), el control de versiones de los procesos y la gestión avanzada de la seguridad y de los perfiles (solicitante, actor, supervisor, gestión de la visibilidad de datos del flujo de trabajo).

El presente video le mostrará cómo diseñar y automatizar sus procesos de negocio a partir de la integración de WorkflowGen con la infraestructura de software de su empresa, tomando como ejemplo los portales cliente y agente de una compañía de seguros.

Integración GraphQL API, Webhooks y Node.js

Con GraphQL los desarrolladores tienen el control de los datos que quieren manipular e intercambiar con el API WorkflowGen. Sea para una aplicación móvil, una aplicación web o micro servicios, ellos tienen exactamente los datos que necesitan con un rendimiento óptimo. El API GraphQL de WorkflowGen ofrece un nuevo nivel de integración y personalización que permite a los desarrolladores extender el esquema GraphQL para agregar sus propios tipos, solicitudes y mutaciones. Con una gran libertad y rapidez, los desarrolladores pueden crear soluciones originales con mucho valor agregado, sobrepasando los esquemas tradicionales de flujos de trabajo y de BPM (manejadores de procesos de negocios) en general para acelerar la transformación digital.

Los webhooks entrantes son soluciones modernas y simples para integrar WorkflowGen con aplicaciones externas. Los desarrolladores pueden crear webhooks entrantes para completar una operación de flujo de trabajo o para crear una nueva solicitud, entre otras operaciones disponibles. Basta con enviar los datos requeridos en formato JSON a un URL único generado por la nueva aplicacion webhooks entrantes de WorkflowGen; no se requiere de una autentificación adicional.


El API GraphQL y los webhooks entrantes son aplicaciones hechas en Node.js ejecutadas como aplicaciones IIS y completamente integradas a la infraestructura WorkflowGen .NET. Este nuevo enfoque híbrido con Node.js se ejecuta sobre del motor de flujo de trabajo .NET y ofrece lo mejor de dos ambientes: integración rápida y personalización con Node.js, en conjunto con algoritmos de BPM, maduros y de alto rendimiento, en .NET.

Infraestructura escalable

WorkflowGen se encarga de las configuraciones de las granjas de servidores web, permitiéndole configurar dos o varios servidores web para manejar las solicitudes de flujo de trabajo. Este tipo de configuración mejora el rendimiento y ofrece una mayor disponibilidad.

Usted puede configurar WorkflowGen para guardar datos de archivos en la base de datos o en el sistema de archivos. El almacenamiento en la base de datos simplifica la administración de los datos de los archivos y permite guardarlos sin interrumpir el servicio. El almacenamiento en un sistema de archivos ofrece las ventajas siguientes :

  • Optimización del tamaño de la base de datos y del servidor de base de datos
  • Reducción del uso del ancho de banda entre el servidor web y el servidor de base de datos
  • Almacenamiento de datos de archivos en una carpeta local o remota




WorkflowGen ofrece varias opciones para optimizar el almacenamiento y el tiempo de respuesta en función de sus prestaciones. Por ejemplo, usted puede reducir la cantidad de datos mostrados en la página principal del portal de usuarios o definir datos de procesos para guardar todas las versiones de los valores de los datos por operación o solamente la última versión de estos.

WorkflowGen ofrece una manera única de mejorar considerablemente el tiempo de respuesta de los procesos gracias a las granjas de bases de datos. Aparte de la base de datos "maestra" WorkflowGen, usted puede configurar una base de datos "esclava" en modo lectura que replique la base de datos "maestra" casi en tiempo real. En esta configuración, WorkflowGen redirige las solicitudes SQL de lectura hacia la base de datos "esclava". Si se toma en cuenta que hay más solicitudes de lectura que de escritura, el rendimiento del producto mejorará considerablemente. Esta mejora en la base de datos es solo aplicable a bases de datos SQL Server.

Editor de formularios de nivel empresarial

Con el editor de formularios web integrado de WorkflowGen, usted podrá crear rápidamente formularios avanzados con funcionalidades empresariales como gestion de tablas, interfaces de usuario multilingües, herramientas personalizables y vistas interactivas de flujo de trabajo, para definir de manera visual el comportamiento del formulario. Las posibilidades de personalización y de integración son posibles gracias a la generación transparente de formularios ASP.NET y a un editor con código oculto de C# integrado.
