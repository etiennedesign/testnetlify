Cree flujos de trabajo usando funcionalidades avanzadas de creación de procesos

Cree flujos de procesos sofisticados a partir de un diseñador gráfico que no necesita de código adicional

Automatice procesos humanos y de sistemas usando un diseñador de procesos fluído y de simple comprensión que utiliza HTML5 para generar múltiples acciones, transiciones, sincronización y creación de subprocesos.

Cree formularios web de manera rápida y sin programar

WorkflowGen dispone de un editor integrado de formularios web que permite a los no-programadores la creación de formularios web profesionales. El editor de formularios está basado en las tecnologias HTML y JavaScript, y no necesita de ningun plugin o software adicional. Si se necesitan funcionalidades o personalizaciones más avanzadas, el editor de formularios contiene un editor de código .NET.

Ofrezca a los participantes de los procesos un portal de flujo de trabajo completo

En una sola página web, los participantes de los procesos pueden crear solicitudes nuevas y monitorear el estado de sus flujos de trabajo en tiempo real. De acuerdo con la configuración de WorkflowGen, el "single sign-on" o "inicio de sesión único" permite una autenticación automática y transparente.

Despliegue flujos de trabajo con funcionalidades de gestión de procesos empresariales

Los diseñadores de procesos pueden organizar los flujos de trabajo en carpetas protegidas para una fácil descentralización y delegar la administración de procesos por carpeta. Los flujos de procesos pueden del mismo modo ponerse a la disposición de los usuarios finales a través de múltiples categorías. WorkflowGen maneja el control de versiones de los flujos de trabajo y permite a los diseñadores de procesos actualizar la definición de los flujos de trabajo en tiempo real.

Apruebe decenas de solicitudes en segundos

La funcionalidad de Aprobación Rápida y en Masa ofrece la posibilidad de tratar decenas de solicitudes en segundos. El usuario puede organizar las solicitudes de acuerdo con los datos de los procesos y crear vistas personalizadas que se pueden mostrar en el portal y de ese modo tener un acceso directo.

Realice operaciones de flujos de trabajo a través de varios procesos en tan sólo unos clicks

En el modo de Vista Rápida, los usuarios solo tienen que cliquear para pasar de un formulario a otro y consultar con precisión el estado de sus solicitudes, realizar sus operaciones con un mínimo de clicks y disfrutar de una navegación avanzada e intuitiva en una sola ventana de vista previa.

Manaje fácilmente las listas globales usadas en los procesos

Los datos relacionados a la empresa, que no son necesariamente manejados por un ERP o un CRM, pueden enriquecer los flujos de trabajo con una administración descentralizada para los que manejan dichos datos. La gestión de listas globales permite a los usuarios autorizados crear y manejar facilmente listas de datos a través del módulo de administración web de WorkflowGen para su uso en diferentes formularios de proceso.

Defina visualmente sus controles con solo unos pocos clicks

La vista interactiva del flujo de trabajo constituye la manera más intuitiva de crear procesos basados en formularios. Permite a los diseñadores de procesos la definición visual del comportamiento del formulario de acuerdo a cada acción del flujo de trabajo. La vista interactiva del flujo de trabajo está integrada de manera transparente con el editor de formularios.

Genere reportes fácilmente

Las funcionalidades de reportes avanzados permiten a los usuarios finales la creación de reportes usando los datos de los procesos.

Ejecute acciones en segundos vía correo electrónico

Los participantes de procesos pueden recibir notificaciones de eventos predeterminados por correo electrónico, y las acciones de flujo de trabajo pueden ser ejecutadas directamente desde notificaciones de correo electrónico en HTML que contienen adjuntos.

Expándase internacionalmente con una solución empresarial multilingüe

Los participantes de procesos pueden cambiar sus preferencias de idioma, zona horaria y contraseña directamente en el portal. La gestión de formularios multilingües y zonas horarias es tambien permitida gracias a la compatibilidad doble-byte y a los archivos de recursos editables.

Manténgase conectado a cualquier dispositivo a partir de applicaciones móbiles intuitivas

La aplicación móvil WorkflowGen permite a los usuarios acceder a las funcionalidades del portal de flujos de trabajo y realizar sus operaciones a distancia a través de sus dispositivos iOS y Android.

Gestión de los participantes

Los administradores pueden fácilmente manejar los participantes de los procesos definiéndolos como usuarios, grupos, directorios o coordinadores. Los participantes globales, del mismo modo que los participantes basados en roles pueden estar activos en varios flujos de trabajo y usar varios perfiles para acceder a las funcionalidades del producto.

Los administradores pueden del mismo modo permitir a los supervisores reasignar acciones, cancelar solicitudes, visualizar estadísticas, limitar la visibilidad de los supervisores de procesos de ciertas solicitudes y autorizar a los usuarios a hacer comentarios en las solicitudes.




Gestion de directorios

Los administradores pueden usar conectores de directorios (Active Directory, archivos CSV, solicitudes nativas LDAP, archivos de texto) para sincronizar a los usuarios.

WorkflowGen incluye funcionalidades empresariales para faciitar la gestión de una gran población de usuarios (sincronización automática de directorios, creación de múltiples directorios, gestión de grupos, gestión de jerarquías, solicitudes en directorios existentes para crear grupos virtuales de manera dinámica, gestión de historial de flujos de trabajo de usuarios borrados, reportes detallados de las actividades de los usuarios, reemplazo de un usuario por otro).




Aplicaciones de flujos de trabajo

WorkflowGen ofrece aplicaciones de flujos de trabajo para automatizar las operaciones de sistemas de procesos, permitiendo así la exportación de datos de un archivo XML hacia una base de datos, la recuperación de informaciones de usuarios de un directorio y el envío de notificaciones a través de mensajes de texto usando la plataforma Twilio, entre otros.

Los desarrolladores pueden crear webhooks, servicios web o assemblies .NET para usarlos como aplicaciones de flujos de trabajo personalizadas que permitan una vasta gama de posibilidades de integración, que incluyen la exportación de datos de procesos y de formularios hacia bases de datos o DMS (software de manejo de datos), el uso de aplicaciones de terceros en los flujos de trabajo, la generación dinámica de flujos de trabajo, la generación dinámica de documentos a partir de datos de flujo de trabajo y la activación de alertas y de notificaciones hacia sistemas de mensajería y de vigilancia.




API Web

La integración con sistemas existentes es fácil con la API GraphQL de WorkflowGen. Los desarrolladores pueden del mismo modo crear sus propios portales de flujo de trabajo o UI/UX especializados usando ambientes de desarrollo existentes.

La mayoría de las operaciones de ejecución y de diseño pueden ser realizadas a través de la API: la recuperación de solicitudes, datos y estados de las operaciones, la creación de solicitudes nuevas, la actualización de las operaciones de flujo de trabajo, la gestión de usuarios, de grupos, de participantes y de delegaciones, así como la importación/exportación de definiciones de procesos.




Autentificación y seguridad

La autenticación en Windows con Active Directory es completamente compatible. Usted puede tambien configurar una autenticación de aplicación (credenciales manejadas en la base de datos) o una autenticación hecha a la medida (soporte de módulos .NET HTTP personalizados, integración con sistemas single sign-on existentes)

WorkflowGen soporta el encriptado HTTPS SSL. La seguridad de las aplicaciones web esta completamente integrada en el proceso de desarrollo del producto de acuerdo con las recomendaciones de la OWASP.




Rendimiento y escalabilidad

WorkflowGen provee un desarrollo altamente escalable diseñado para un uso intensivo. La estructura y el tamaño de las bases de datos están optimizadas y los archivos de datos son guardados usando el sistema de archivos o un sistema de base de datos.

Los administradores pueden instalar o desplegar WorkflowGen en un ambiente de "Web Farm" usando balanceo de carga y usar múltiples servidores para las aplicaciones web. Se pueden usar instancias adicionales de bases de datos (replicadas en tiempo real) para tratar solicitudes SQL con el fin de reducir el tiempo de respuesta del servidor (escalabilidad)
