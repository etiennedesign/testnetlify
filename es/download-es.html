Descargar la versión gratuita














Resumen de elementos técnicos

La version gratuita incluye todas las funcionalidades y le permitirá implementar una versión de un proceso para usuarios ilimitados en una instancia. No hay límite de tiempo y la versión es gratuita para siempre.


  • Aplicación Web .NET ejecutada en Windows Server (2008, 2012, 2016) o Windows 10 (ambientes de desarrollo).

  • Bases de datos compatibles : MS SQL (2008, 2012, 2014, 2016, Express Edition) y Oracle (10g, 11g y R2, 12c).

  • Completamente integrado con IIS, con soporte integrado para la autentificación transparente de Windows (Active Directory) (otros métodos de autentificación están también disponibles).
  • Usa el gateway IIS SMTP o un servidor SMTP existente para enviar las notificaciones por correo electrónico.

  • La configuración incluye la instalación de una aplicación Web en .NET y la instalación de un servicio de Windows.

  • Los archivos de datos de la aplicación son guardados en el servidor Web por defecto.
