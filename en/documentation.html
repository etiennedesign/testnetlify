WorkflowGen Setup and Technical Configuration

The WorkflowGen Technical Reference Guide covers all of the procedures involved in the installation, setup, and maintenance of WorkflowGen. It is aimed at system administrators who will be required to install or carry out maintenance of the application.

WorkflowGen Features and Technical Requirements

This guide provides a detailed list of WorkflowGen features and technical requirements, as well as information on configurations and scalability, and examples of usage metrics.


Version 7.10.0   EN / FR

WorkflowGen Process Design and Administration

The WorkflowGen Administration Module Reference Guide describes all the screens of the WorkflowGen Administration Module and provides instructions for the form designer, the workflow designer, and user and participant management, as well as a complete list of related operating rules. It also describes the built-in workflow applications and how to use them, and includes a list of possible errors along with their explanations.

WorkflowGen Portal

The WorkflowGen User Portal Reference Guide provides instructions on how to use WorkflowGen Portal, including how to launch, handle, follow-up and delegate requests, perform searches using Advanced View, use the Quick Approval feature, generate real-time statistics, and create custom charts and dashboards. The guide also includes a list of possible errors along with their explanations. It is an important resource for process participants.
